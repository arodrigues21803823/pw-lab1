CSS

Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web.

Tipos de Seletores
Elemento Utilizou? Como funciona?
Universal done aplica o estilo a todos os elementos
Tipo done aplica o estilo aos marcadores de certo tipo
Classe done aplica o estilo a uma classe
Identificador done aplica o estilo a um item específico
Atributos close aplica o estilo consoante o atributo de um elemento
Pseudo-classe close aplica o estilo consoante o estado de um elemento
Pseudo-elemento close aplica o estilo consoante as partes de elementos

Combinadores de Seletores
Elemento Utilizou? Como funciona?
Descendentes done div p – todos os p que estão “dentro” de um div
Filhos done div > p – todos os p que estão “logo dentro” de div
Adjacentes done div ~ p – todos os p que estão colocados depois de um div
Imediatamente adjacentes done div + p – todos os p que estão colocados “imediatamente” depois de um div
Agrupação done div,p{ color: red; } - os div e p ficacarão com fonte vermelha
